# Conventions

- Fichiers et dossiers : minuscules avec tirets.
- Dates : format ISO `AAAA-MM-JJ`.
- Versions : identifiant officiel, période ou date de publication.
- Contenu public : français clair, termes techniques expliqués.
- Conclusions : distinguer corrélation, causalité et incertitude.

Exemples de commits :

```text
Ajouter l'analyse sur le coût du logement
Mettre à jour les données INSEE 2026-06
Corriger l'intégration du dashboard Tableau
```
